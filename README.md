# Ashutosh Kumar Tiwari

A personal portfolio and blog built with React, deployed using GitHub Pages.

---

This repository hosts the source code for my personal website, which showcases my projects, blog posts, and contact information. The site was bootstrapped with [Create React App](https://create-react-app.dev/) and is configured to publish automatically to the `gh-pages` branch whenever updates are pushed to `main`.

## Table of Contents

- [Features](#features)  
- [Tech Stack](#tech-stack)  
- [Author](#author)  

## Features

- **React Single-Page App** with client-side routing  
- **Responsive Design** for desktop and mobile  
- **Portfolio Gallery** to highlight projects and case studies  
- **Blog Section** powered by Markdown files  
- **Contact Form** (via third-party form service)  
- **SEO Optimized** metadata and sitemap  

## Tech Stack

- **Framework:** React (Create React App)  
- **Language:** JavaScript (ES6+)  
- **Styling:** CSS Modules / plain CSS  
- **Deployment:** GitHub Pages (`gh-pages` npm package)
